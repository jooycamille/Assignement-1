Enhancements

  1. Figure alignment with the caption:

    figure {
    text-align: center;
    display: table;
    }

    figcaption {
    display: table-caption;
    caption-side: bottom;
    }
    https://www.w3.org/Style/Examples/007/figures.en.html

      Extension Application:
    1. development image
    2. growth and future image
    3. relevant technology image

  2. Zoom on Hover:

    #g_img:hover {
    transform: scale(1.5);
    }
    https://www.w3schools.com/howto/howto_css_zoom_hover.asp

      Extension Application:
    1. Hover on photo to see it transform

  3. Side Hoverable Navigation Buttons:

    #sidenav a {
    position: fixed;
    left: -220px;
    transition: 0.3s;
    padding: 20px 40px 20px;
    width: 260px;
    text-decoration: none;
    font-size: 16px;
    text-transform: uppercase;
    color: black;
    border-radius: 0 20px 20px 0;
    margin: 0;
    }

    #sidenav a:hover {
    left: 0;
    background-color: #000000;
    color: #e6d1ee;
    }
    https://www.w3schools.com/howto/howto_css_sidenav_buttons.asp

      Extension Application:
    1. Side Navigation

  4. Create Tabbed Menu:

    .q_sect article {
    display: none;
    }

    .q_sect article:target {
    display: block;
    }
    https://www.w3schools.com/cssref/sel_target.asp

      Extension Application:
    1. Quiz Navigation Tabs for the questions

  5. Word Wrap:

    cite {
    word-wrap: break-word;
    }
    https://www.w3schools.com/cssref/css3_pr_word-wrap.asp

      Extension Application:
    1. Resize to see the citations word-wrap as you decrease the size
